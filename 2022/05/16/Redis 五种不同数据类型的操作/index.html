Redis 五种不同数据类型的操作

发表于2022-05-16|更新于2022-05-16|Redis
|字数总计:3.3k|阅读时长:17分钟|阅读量:

Redis 五种不同数据类型的操作

一、Redis 键

(1) keys *

查看当前DB所有的键

*可以替换成?，*代表多个字符，?代表一个字符

(2) exists key

判断某个key是否存在

(3) move key db

将某个key移动到不同的DB下（Redis默认有16个数据库）

(4) expire [key] [time]

设置过期时间

二、String

(1) del/set/get/append/strlen

(2) incr/decr/incrby/decrby

字符串型，对value +1 -1的操作，应用场景多在于浏览数，点赞数，访问数

(3) getrange/setrange

getrange: 类似于python的切片，但是终点不会像python一样不输出

setrange: 从设置的起点位置开始往后覆盖，超出原先范围后新增

(4) setex/setnx

setex: 给key设置过期时间，与expice不同的地方在于，可以设置不存在的key，创建新key

setnx: 在指定的 key 不存在时，为 key 设置指定的值。

(5) mset/mget/msetnx

set,get,setnx 前加 m 表示一次执行多个key，若其中有一个key不成功，则全体失败

三、List

flushdb 用于清空当前数据库中的所有 key。

(1) lpush/rpush/lrange

lpush : 头插法

push : 尾插法

lrange : 输出start到end list 中的值

(2) lpop/rpop

lpop : 输出头元素并删除

rpop : 输出尾元素并删除

(3) lindex

lindex : 按照索引下标获得元素

(4) llen

llen : 返回列表长度

(5) lrem

lrem : 根据参数 COUNT 的值，移除列表中与参数 VALUE 相等的元素。

COUNT 的值可以是以下几种：

  • count > 0 : 从表头开始向表尾搜索，移除与 VALUE 相等的元素，数量为 COUNT 。
  • count < 0 : 从表尾开始向表头搜索，移除与 VALUE 相等的元素，数量为 COUNT 的绝对值。
  • count = 0 : 移除表中所有与 VALUE 相等的值

四、Set

（1）sadd/smembers/sismember

sadd: 创建一个集合

smembers:查看集合元素

sismember:查找集合中是否存在某元素

（2）scard

scard:获取集合中的元素个数

（3）srem

srem:删除集合中元素

（4）srandmember

srandmember:随机取出几个数

（5）spop

spop:随机出栈（原数组删除）

（6）smove

smove:移动集合中的元素，前给后

（7）sdiff/sinter/sunion

sdiff:差集

sinter:交集

sunion:并集

五、Hash

（1）hset/hget/hmset/hmget/hgetall/hdel

hset: 用于为哈希表中的字段赋值，格式HSET KEY_NAME FIELD VALUE，如果字段已经存在于哈希表中，旧值将被覆盖。

hget:查看哈希表中字段的值

hmset:为哈希表多字段赋值

hmget:查看哈希表中多字段的值

hgetall:返回所有的字段和值

hdel:删除某字段对应的值

（2）hlen

hlen:查看哈希表长度

（3）hexists

hexists: 查看哈希表中某字段是否存在

（4）hkeys/ hvals

hkeys: 查看哈希表中所有的字段

hvals: 查看哈希表中所有的值

（5）hincrby / hincrbyfloat

hincrby:用于为哈希表中的字段值加上指定增量值，也就是对指定字段进行减法操作。

hincrbyfloat:命令用于为哈希表中的字段值加上指定浮点数增量值。

如果指定的字段不存在，那么在执行命令前，字段的值被初始化为 0 。

（6）hsetnx

hsetnx:用于为哈希表中不存在的的字段赋值 。

六、Zset（sorted set）

在set基础上，加一个score值

之前set是 k1 v1 v2 v3，

现在zset是 k1 score1 v1 score2 v2

（1）zadd/zrange

zadd: **创建一个Zset，格式zadd zset_name score value [score value...]**

zrange: 查看Zset

（2）zrange by scorekey

zrangebyscorekey:返回有序集合中指定分数区间的成员列表。

（3）zrem

zrem : 删除元素

（4）zcard/zcount/zrank/zscore

zcard: 获取共有元素

zcount: 获取区间中一共有多少元素

zrank: 获取下标值

zscore: 获得分数

（5）zrevrank

zrevrank: 获得逆序下标值

（6）zrevrange

zrevrange: 逆序输出

（7）zrevrangebyscore

zrevrangebyscore: 逆序返回有序集合中指定分数区间的成员列表。
