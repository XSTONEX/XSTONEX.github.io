Redis 不同数据类型的操作

一、Redis 键

(1) keys *

查看当前DB所有的键

*可以替换成?，*代表多个字符，?代表一个字符

(2) exists key

判断某个key是否存在

(3) move key db

将某个key移动到不同的DB下（Redis默认有16个数据库）

(4) expire [key] [time]

设置过期时间

二、String

(1) del/set/get/append/strlen

(2) incr/decr/incrby/decrby

字符串型，对value +1 -1的操作，应用场景多在于浏览数，点赞数，访问数

(3) getrange/setrange

getrange: 类似于python的切片，但是终点不会像python一样不输出

setrange: 从设置的起点位置开始往后覆盖，超出原先范围后新增

(4) setex/setnx

setex: 给key设置过期时间，与expice不同的地方在于，可以设置不存在的key，创建新key

setnx: 在指定的 key 不存在时，为 key 设置指定的值。

(5) mset/mget/msetnx

set,get,setnx 前加 m 表示一次执行多个key，若其中有一个key不成功，则全体失败

三、List

flushdb 用于清空当前数据库中的所有 key。

(1) lpush/rpush/lrange

lpush : 头插法

push : 尾插法

lrange : 输出start到end list 中的值

(2) lpop/rpop

lpop : 输出头元素并删除

rpop : 输出尾元素并删除

(3) lindex

lindex : 按照索引下标获得元素

(4) llen

llen : 返回列表长度

(5) lrem

lrem : 根据参数 COUNT 的值，移除列表中与参数 VALUE 相等的元素。

COUNT 的值可以是以下几种：

  • count > 0 : 从表头开始向表尾搜索，移除与 VALUE 相等的元素，数量为 COUNT 。
  • count < 0 : 从表尾开始向表头搜索，移除与 VALUE 相等的元素，数量为 COUNT 的绝对值。
  • count = 0 : 移除表中所有与 VALUE 相等的值
