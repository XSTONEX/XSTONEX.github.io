==①如何计算二维前缀和矩阵:==
Sx,y = Sx-1,y + Sx,y-1 - Sx-1,y-1 + ax,y

==②如何利用二维前矩阵和，计算子矩阵和：==
S[x1,y,x2,y2] = Sx2,y2 - Sx2,y1-1 - Sx1-1,y2 + Sx1-1,y1-1

原题链接

K倍区间

原题链接

Ai,Ai+1,…AjAi,Ai+1,…Aj 之和是 K 的倍数，我们就称这个区间 [i,j]是 K 倍区间

求有多少个K倍区间

解法：一维前缀和 + 数学优化

优化原理：(sum[r] - sum[l-1])%k == 0 == sum[r]%k == sum[l-1]%k
